Working Memory · Daily Practice

毎日のN-Backで、
ワーキングメモリを鍛える。

認知心理学の研究で広く使われるN-Backタスクに、毎日取り組めるシンプルな訓練ツール。脳トレ「ゲーム」ではなく、計測と継続のための「計器」として設計しました。

特徴を見る プライバシー

特徴

余計な装飾を削ぎ落とし、訓練と記録だけに集中できる設計。

01

位置 + 音声のデュアルN-Back

3×3グリッドの位置と読み上げ音の2モードを同時にチェック。研究で使われる標準的な形式に準拠。

02

毎日の記録とグラフ

日々のスコア・正答率・N値の推移をSwift Chartsで可視化。継続を支える計測ダッシュボード。

03

iCloud自動同期

SwiftData + CloudKitで複数デバイス間のデータを自動同期。機種変更後も記録が引き継がれます。

04

Game Center対応

3種のリーダーボードと8種のアチーブメント。自分との対話に加えて、世界との比較も。

05

訓練中は広告ゼロ

集中を妨げないため、トレーニング中は広告を一切表示しません(鉄則)。

06

日本語 / 英語対応

UI・音声キューともに日英2言語に対応。端末の言語設定に自動追従します。
